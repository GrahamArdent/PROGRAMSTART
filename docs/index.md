# PROGRAMSTART

PROGRAMSTART is a documentation-first workflow platform for product planning, validation, and launch discipline.

Use the primary source files in the repository for the authoritative workflow content:

- `README.md`
- `QUICKSTART.md`
- `PROGRAMBUILD/PROGRAMBUILD_CANONICAL.md`
- `PROGRAMBUILD/PROGRAMBUILD_FILE_INDEX.md`

This site exists to give the platform a searchable entry point, not to replace the source-of-truth markdown files.

Additional platform reference pages in this site:

- `context-layer.md` explains the hybrid context model and the generated context index/query surface
- `knowledge-base.md` captures the research-backed stack, platform, workflow, and retrieval guidance now indexed by the context layer
- `knowledge-ops.md` defines how weekly deep-research updates should be run so the KB stays decision-oriented instead of turning into an archive
- `dashboard-api.md` documents the local dashboard's internal HTTP contract and command whitelist model
- `reusable-patterns.md` captures the patterns that should transfer between PROGRAMSTART and the programs it helps produce
- `defaults-review.md` records which USERJOURNEY ideas should become PROGRAMBUILD defaults and which should remain optional attachments
